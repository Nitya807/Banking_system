# Banking-system-
A simple CLI-based banking system in Python that allows users to deposit, withdraw, and manage transactions efficiently.
#  Python Banking System

##  Project Description
The Python Banking System is a simple command-line application that simulates basic banking operations such as depositing money, withdrawing money, checking balance, and viewing transaction history. This project is designed for beginners to understand Object-Oriented Programming (OOP) concepts in Python.

---

##  Objectives
- Learn and implement OOP concepts in Python
- Simulate real-world banking operations
- Manage user transactions efficiently
- Maintain transaction history

---

##  Features
-  Account creation (Name & Account Number)
-  Deposit money
-  Withdraw money (with balance validation)
-  Check current balance
-  View transaction history
-  Loading animation for better user experience

---

##  Technologies Used
- **Python**
- Object-Oriented Programming (OOP)
- `time` module (for loading effect)
- Lists (for storing transaction history)
